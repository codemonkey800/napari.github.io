Plugins¶

napari enables developers to extend functionality of the program by writing plugins. For a guide on how to create a plugin, see creating a napari plugin. For napari contributors looking to understand how the napari plugin architecture is implemented, see napari plugin architecture. For a complete list of hook specifications that developers can implement, see the napari hook specification reference.
